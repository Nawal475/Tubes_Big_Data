# Dataset Documentation

## Dataset 1
Nama Dataset:
Olist Brazilian E-Commerce Dataset

Sumber:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Deskripsi:
Dataset e-commerce Brasil yang berisi informasi pesanan, pelanggan, produk, pembayaran, dan ulasan pelanggan. Dataset ini digunakan sebagai sumber utama proses ETL dan ELT.

## Dataset 2
Nama Dataset:
BRL/USD Exchange Rate Dataset

Sumber:
Frankfurter API

Link:
https://www.frankfurter.app/

Deskripsi:
Dataset nilai tukar BRL terhadap USD periode 2016–2019 yang digunakan untuk mengonversi nilai transaksi dari mata uang BRL ke USD dalam proses analisis data warehouse.
